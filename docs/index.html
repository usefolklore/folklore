§01 · MANIFESTO [DECLARED]

We compound on inference. Each peer supplies the network with reasoning it has already done, so the next peer infers from there, not from zero. Knowledge builds on knowledge. The network reasons deeper than any node alone, and nobody starts from square one.

A cooperative knowledge protocol. Each peer keeps only its own graph: what it has read, debugged, fetched, signed. It queries the graphs of every other peer.

Knowledge that used to take minutes now arrives in milliseconds. Off the grid. On the commons.

This is the next advancement in human knowledge and consciousness: working together to reach better, deeper, fuller understanding. Never in the history of mankind has cooperation between minds been this seamless.

And the more the network is asked, the more certain it grows. Every query enriches both graphs: the asker’s and the answerer’s. Compounding inference is compounding confidence.

Welcome to the next generation of the web.
The cooperative web.

§04 · PROVENANCE [SIGNED]

Anonymous retrieval is a poisoning surface.

Every other AI memory feeds your model unsigned text from nowhere.

RAG’s open wound

5 → 97%

5 poisoned documents can hijack a retrieval system with a 97% attack success rate. The model has no way to tell trustworthy context from an injected lie.

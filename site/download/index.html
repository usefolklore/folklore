Get folklore.

One app for every desk: it installs the memory node, wires every AI coding tool on your machine, and starts the daemon. No terminal needed.

macOS.dmg · signed & notarized↓ Windows.exe installer↓ Linux.AppImage · .deb↓

After the download

macOS

  1. Open the .dmg, drag Folklore to Applications.
  2. Launch it. Builds are signed and notarized by Apple: no security prompt.
  3. Click Install everything in the wizard.

macOS 13 Ventura or newer. Universal binary (Apple Silicon + Intel).

Windows

  1. Run the .exe (or .msi for managed installs).
  2. Builds are unsigned for now. If SmartScreen warns: More info, then Run anyway.
  3. Click Install everything in the wizard.

Windows 10 or newer, 64-bit.

Linux

  1. AppImage: chmod +x Folklore*.AppImage, then run it.
  2. Debian/Ubuntu: sudo dpkg -i Folklore*.deb
  3. Click Install everything in the wizard.

The tray icon needs libayatana-appindicator on some distros.

Prefer the command line?

$ npm install -g @usefolklore/folklorecopy
$ brew install --cask usefolklore/tap/folklorecopy

npm installs the node + CLI only (bring your own Node 20+). The Homebrew cask installs the same desktop app as the macOS button above.

Known issues

WindowsBuilds are not code-signed yet, so SmartScreen shows a warning on first run. Choose More info, then Run anyway. Signing is planned.
AllFirst launch fetches the folklore CLI with the bundled Node runtime, so the very first Install everything needs a network connection. Later runs work offline.
LinuxOn distros without libayatana-appindicator the tray icon may not appear; install it and relaunch. The wizard window works regardless.
macOSVersion 0.1.0 predates signing and will show a Gatekeeper prompt. Use 0.1.1 or newer instead.

Hit something new? Open an issue: every report sharpens the next release.
